# 实训项目-JavaWeb仿小米在线商城ShoppingMall

#### 介绍
JavaWeb在线商城开发知识总结(java+jsp+servlet+MySQL+jdbc+css+js+jQuery)
这个基于JavaWeb的在线商城项目是我们专业大三上学期期末实训的项目
做的不是很好，但是基本功能能够实现
